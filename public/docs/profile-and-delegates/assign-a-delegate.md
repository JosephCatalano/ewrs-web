# Assign a Delegate

A delegate is a person who can book a workspace on behalf of an employee.

> **Reminder:** Make sure the person knows you assigned them as your delegate in EWRS.

## Steps

1. Log into the EWRS homepage.
2. Select the drop-down arrow next to your name in the top-right corner.
3. Select **Edit Account**.
4. Under **Assign Delegates**, enter the name of the person you want to assign.

![Assign Delegates search field](/docs/assets/ewrs-manual/assign-delegate-search.jpg)

5. Select the person from the list.
6. Select **Save**.

![Profile updated successfully message](/docs/assets/ewrs-manual/profile-updated-message.png)

7. EWRS returns you to the Home page after the profile is updated.

## Delegate limit

| Rule              | Limit |
| ----------------- | ----- |
| Maximum delegates | 5     |
