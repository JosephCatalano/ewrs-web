# Remove a Delegate

Remove a delegate from **Edit Account** when that person should no longer book workspaces on your behalf.

## Steps

1. Log into the EWRS homepage.
2. Select the drop-down arrow next to your name in the top-right corner.
3. Select **Edit Account**.
4. Under **Assign Delegates**, select **X** next to the delegate you want to remove.
5. Select **Save**.

![Profile updated successfully message](/docs/assets/ewrs-manual/profile-updated-message.png)

6. EWRS returns you to the Home page after the profile is updated.

> **Note:** Removing a delegate stops that person from making future bookings on your behalf.
