# Edit Your Profile

Use **Edit Account** to update your reporting manager, division, delegates, and theme preference.

## Steps

1. Log into the EWRS homepage.
2. Select the drop-down arrow next to your name in the top-right corner.

![Profile menu with Edit Account and theme options](/docs/assets/ewrs-manual/profile-menu.jpg)

3. To change the theme, select **Set Dark Mode** or **Set Light Mode**, depending on your current theme.
4. Select **Edit Account**.
5. On the **Update Profile** page, update your profile fields.

![Update Profile page](/docs/assets/ewrs-manual/update-profile-page.jpg)

6. Select **Save**.

![Profile updated successfully message](/docs/assets/ewrs-manual/profile-updated-message.png)

7. EWRS returns you to the Home page after the profile is updated.

## Profile fields

| Field | Required | Details |
| --- | --- | --- |
| Reporting Manager | Yes | Select your manager |
| Division | Yes | Select your division |
| Delegates | No | Add or remove delegates |

> **Note:** Fields marked with an asterisk are mandatory.

## Theme preference

EWRS supports light and dark modes. When you first access the application, EWRS automatically matches your device's system theme preference. You can manually switch the theme from the profile menu.
