# Check Out

You must check out when you are finished using your workspace.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select the **My Bookings** tab. Your bookings will be displayed.
3. Find the booking you want to check out of.
4. Select **Check Out**.
5. Select **Yes** to check out of the booking.

![Check out confirmation dialog](/docs/assets/ewrs-manual/checkout-confirmation.png)

6. You have now checked out of your workspace.

> **Reminder:** Check out when you are finished so workspace usage is recorded accurately.
