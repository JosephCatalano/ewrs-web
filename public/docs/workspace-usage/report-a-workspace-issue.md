# Report a Workspace Issue

Use **Report an Issue** from **My Bookings** to report a problem with a workspace.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select the **My Bookings** tab.
3. Scroll to the bottom of the page to find the **Report an Issue** bar.

![Report an Issue bar at the bottom of My Bookings](/docs/assets/ewrs-manual/report-issue-bar.png)

4. Select the drop-down list to expand the issue form.
5. Complete the form.

![Report an issue form](/docs/assets/ewrs-manual/report-issue-form.png)

6. Select **Send**.

![Issue report sent message](/docs/assets/ewrs-manual/report-sent-message.png)

## Required fields

| Field       | Required |
| ----------- | -------- |
| Location    | Yes      |
| Floor       | Yes      |
| Desk No.    | Yes      |
| Description | Yes      |

> **Note:** Fields marked with an asterisk are mandatory.
