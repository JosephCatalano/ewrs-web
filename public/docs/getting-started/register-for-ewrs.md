# Register for EWRS

Register the first time you access EWRS so the system can associate your profile with your division, manager, and any delegates.

## Steps

1. Log into the EWRS homepage.
2. Select **Register**.
3. Select your **Reporting Manager** and **Division** from the drop-down lists.

![EWRS registration page showing reporting manager, division, and delegate fields](/docs/assets/ewrs-manual/register-home.jpg)

4. Optionally assign up to five delegates. Enter the delegate's name and select the person from the drop-down list.

![Assign delegates field on the registration page](/docs/assets/ewrs-manual/register-delegates.jpg)

> **Note:** A delegate is a person who can book a workspace on behalf of an employee.

5. Review the declaration and select the checkbox to agree.

![Registration declaration checkbox](/docs/assets/ewrs-manual/register-declaration.jpg)

> **Note:** Your Building Procedures Manual will be sent to you by email after you confirm a booking.

6. Select **Register** to enter the booking system.

## Registration fields

| Field | Required | Details |
| --- | --- | --- |
| Reporting Manager | Yes | Select from the drop-down list |
| Division | Yes | Select from the drop-down list |
| Delegates | No | Add up to five delegates |
| Declaration | Yes | Must be accepted before registration |
