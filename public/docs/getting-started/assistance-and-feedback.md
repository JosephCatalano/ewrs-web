# Assistance and Feedback

For technical inquiries, issues, or suggestions, contact EWRS support by email:

| Support channel | Details |
| --- | --- |
| Email | EWRSfeedback@ontario.ca |
| In-app link | Select **Contact Us** at the bottom of the EWRS page |

## Access issues

If you cannot access EWRS, or you do not see the registration page after logging in, you may not have been added to the EWRS system yet. EWRS will show an error message with next steps.

To request access, email EWRSfeedback@ontario.ca and include:

| Required information | Description |
| --- | --- |
| Full name | Your legal or work profile name |
| Email address | Your OPS email address |
| Division | Your division name |
| Manager's name | Your reporting manager |
