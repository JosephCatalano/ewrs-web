# Introduction

The Employee Workspace Booking System (EWRS) is the online booking tool Ontario Public Service employees use to reserve a workspace.

Use EWRS to book a workspace for the same day or up to 28 days in advance at your main location or locations.

| Booking type                 | Availability                          |
| ---------------------------- | ------------------------------------- |
| Standard workspace locations | Same day and up to 28 days in advance |
| OPS Touchdown Centres (OTC)  | Up to 7 days in advance               |
| Booking days                 | Monday to Friday                      |

> **Note:** OPS Touchdown Centre bookings can only be made 7 days in advance.

![28 Day Booking Window](https://ewrs0001prd0420pub.blob.core.windows.net/ewrs-assets/28-day.gif)

If you make a booking and no longer need the workspace, cancel it in EWRS as soon as possible so another employee can book it.

## First-time access

First-time users must register in EWRS. Registration includes selecting your division, identifying your reporting manager, and optionally adding delegates who can book workspaces on your behalf.
