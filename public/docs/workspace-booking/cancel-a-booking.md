# Cancel a Booking

Bookings can be cancelled at any time.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select the **My Bookings** tab. Your bookings will be displayed.
3. Find the booking you want to cancel.
4. Select **Cancel** on the right side of the page.

![My Bookings tab showing a cancel button](/docs/assets/ewrs-manual/my-bookings-cancel.png)

5. Select **Yes** to cancel the booking.

![Cancel booking confirmation dialog](/docs/assets/ewrs-manual/cancel-confirmation.png)

6. Review the cancellation email confirmation.

![Workspace cancellation confirmation email](/docs/assets/ewrs-manual/cancellation-email.png)

> **Important:** Cancelling the Outlook calendar invitation does not cancel the booking in EWRS. You must cancel the booking from the EWRS booking system.

## After cancellation

| Item               | Action                                                         |
| ------------------ | -------------------------------------------------------------- |
| EWRS booking       | Cancelled in EWRS                                              |
| Calendar event     | Automatically updated to show "[CANCELLED]" in the event title |
| Confirmation email | Keep as confirmation that the workspace was released           |

> **Note:** Your Outlook calendar event is automatically updated when you cancel a booking in EWRS. The event title will be updated to include "[CANCELLED]", so your calendar stays in sync with EWRS without manual updates.
