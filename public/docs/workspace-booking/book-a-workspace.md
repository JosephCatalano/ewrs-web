# Book a Workspace

Workspaces can be booked for the same day and up to 28 days in advance at non-OTC locations. Workspace bookings are available Monday to Friday.

> **Note:** OPS Touchdown Centre locations cannot be booked for multiple repeat days.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, go to the **Make Booking** section.

![Make Booking section on the EWRS home page](/docs/assets/ewrs-manual/make-booking-section.png)

3. Select the desired **Location** from the drop-down list.

![Location drop-down list](/docs/assets/ewrs-manual/location-dropdown.png)

4. Select the desired **Floor(s)** from the drop-down list. You may select multiple floors.

![Floor selection drop-down list](/docs/assets/ewrs-manual/floor-selection.png)

5. Select the desired **Time Period** from the drop-down list.

![Time period drop-down list](/docs/assets/ewrs-manual/time-period-dropdown.png)

6. Select the **Start Date** you want to book.

![Start date picker](/docs/assets/ewrs-manual/start-date-picker.png)

> **Note:** If you are booking for one day only, you do not need to select **Repeat** or choose an end date.

7. If you are booking for multiple days, select **Repeat**, choose an **End Date**, and optionally choose recurrence days.

![Repeat booking options with end date and weekday checkboxes](/docs/assets/ewrs-manual/repeat-booking-options.png)

8. Select **Search**.

![Search warning and search action area](/docs/assets/ewrs-manual/search-warning.png)

9. Review the search results.

![Search results showing available workstations](/docs/assets/ewrs-manual/search-results.png)

10. Select **Book** for the workstation you want.
11. Select **Yes** to confirm the workstation and timeframe.

![Booking complete message](/docs/assets/ewrs-manual/booking-complete-message.jpg)

## Search results

The results table includes:

| Column | Description |
| --- | --- |
| Desk number | The workstation identifier |
| Floor number | The floor where the workstation is located |
| Features | Available workspace equipment |

## Workspace features

| Feature | Meaning |
| --- | --- |
| Monitor(s) | The workspace includes one or more monitors |
| Mouse | The workspace includes a mouse |
| Keyboard | The workspace includes a keyboard |

> **Tip:** Hover over feature icons in EWRS to see what each icon represents.

## Confirmation email

After the booking is confirmed, you will receive one consolidated Outlook email containing your workspace booking confirmation, calendar invitation, and building procedures. Accept the calendar invitation to save it to your Outlook calendar and review the building

![Consolidated workspace booking confirmation email-1](/docs/assets/ewrs-manual/outlook-calendar-invite-1.png)

## Auto-Fill Booking

When you make a new booking, EWRS remembers your previous search preferences. The next time you visit the booking section, your location, floor, and time period will be automatically pre-filled based on your last booking. This lets you complete bookings faster without re-entering the same information each time.

![Auto Fill gif](https://ewrs0001prd0420pub.blob.core.windows.net/ewrs-assets/autofill.gif)

> **Tip:** If you want to start with different preferences, simply modify the pre-filled information before searching.
