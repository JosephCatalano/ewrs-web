# View Floor Plans

Floor plans are available from each building card.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select **Display Buildings**.
3. Select the arrow on the building card.
4. Choose a floor plan from the drop-down list.

![Floor plan drop-down list for a building](/docs/assets/ewrs-manual/floor-dropdown.png)

5. The floor plan opens in a new browser tab.

![Example floor plan for a building](/docs/assets/ewrs-manual/floor-plan.jpg)

> **Note:** If your browser blocks new tabs, allow pop-ups for EWRS and try again.
