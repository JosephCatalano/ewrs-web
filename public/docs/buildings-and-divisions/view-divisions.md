# View Divisions

Use **Manage Divisions** to look up division and division administrative information.

## Steps

1. Log into the EWRS homepage.
2. Select the **Manage Divisions** tab.

![Manage Divisions tab in EWRS](/docs/assets/ewrs-manual/manage-divisions-tab.png)

3. Type a ministry or division name to view division and division administrative information.

![Manage Divisions search results](/docs/assets/ewrs-manual/manage-divisions-search.png)

## Search options

| Search by     | Result                                           |
| ------------- | ------------------------------------------------ |
| Ministry name | Shows matching ministry and division information |
| Division name | Shows matching division information              |
