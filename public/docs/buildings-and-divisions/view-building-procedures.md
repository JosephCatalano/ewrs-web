# View Building Procedures

Building procedures are available from the same building card area as floor plans.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select **Display Buildings**.
3. Select the arrow on the building card.
4. Choose a building procedure from the drop-down list.

![Building procedure drop-down list for a building](/docs/assets/ewrs-manual/building-procedure-dropdown.png)

5. The building procedure downloads to your device.

> **Reminder:** Review the building procedure before using a workspace at that location.
