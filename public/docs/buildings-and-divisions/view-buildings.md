# View Buildings

Use **Display Buildings** to view available buildings and their amenities.

## Steps

1. Log into the EWRS homepage.
2. From the Home page, select **Display Buildings**.

![Home page with Display Buildings highlighted](/docs/assets/ewrs-manual/home-display-buildings.png)

3. Review the building cards.

![Building cards showing available buildings and amenities](/docs/assets/ewrs-manual/building-cards.jpg)

## Building amenities

The icons at the bottom of each building card represent amenities available near the building.

| Amenity              | Meaning                                     |
| -------------------- | ------------------------------------------- |
| Pay-per-use parking  | Paid parking is available near the building |
| Daily parking        | Daily parking is available                  |
| Accessibility access | Accessibility access is available           |
| Direct access to TTC | Direct access to TTC is available           |

> **Tip:** Hover over an amenity icon in EWRS to see what it represents.
